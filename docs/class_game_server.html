Detailed Description

This class represents a game server.

The server starts automatically and loads the player prefab into the online scene.

@source SWTP-Framework Modified by: Christopher-Marcel Klein

Handles the results of the game and sets the readyToQuit flag. Needs the placement of the local player and the name of the winning player and writes them into a JSON that is used to give rewards in the framework. IMPORTANT: THIS HAS TO BE CALLED AT THE END OF THE GAME!

◆ Start()

void GameServer.Start ( )
private

Start is called before the first frame update.

The server loads the player info of the local player, if:

  • the player is the host -> start server as host
  • the player is the client -> join server as client NOTE: This is not important for development.

@source SWTP-Framework

◆ Update()

void GameServer.Update ( )

Update is called once per frame.

Checks if the local player is a client and not connected. If so the client tries to connect and the disconnect timer is updated.
