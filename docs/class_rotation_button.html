void RotationButton.Awake ( )

The button for rotation is stored in a prefab.

This is then used to find out the name of the prefab and store it in a global variable. This is then accessible from everywhere. In addition, the rotation of the prefab is set to zero parentPanel: GameObject of the panel by the rotation button
